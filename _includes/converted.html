倒立摆仿真与LQR控制实现

本文系统介绍了倒立摆系统的物理建模、LQR控制器设计、数值仿真与可视化动画。内容涵盖了从数学建模、状态空间表达、线性化分析，到Python代码实现的全过程，适合控制理论学习、仿真实验与工程实现参考。

一. 物理建模

1. 数学物理模型

2. 从拉格朗日方程说起

  • 动能：

    T=12Mx˙2⏟小车动能+12m(x˙2+l2θ˙2+2lx˙θ˙cos⁡θ)+12112m(2l)2θ˙2⏟杆的动能

  • 总势能：

    V=mglcos⁡θ

  • 拉格朗日函数：

    L=T−V=12(M+m)x˙2+23ml2θ˙2+mlx˙θ˙cos⁡θ−mglcos⁡θ

3. 运动方程推导

  • 对 x 的欧拉-拉格朗日方程：

    ddt(∂L∂x˙)−∂L∂x=F⇒(M+m)x¨+mlθ¨cos⁡θ−mlθ˙2sin⁡θ=F

  • 对 θ 的欧拉-拉格朗日方程：

    ddt(∂L∂θ˙)−∂L∂θ=0⇒43ml2θ¨+mlx¨cos⁡θ−mglsin⁡θ=0

4. 状态空间表达式（构建仿真器）

  • 矩阵化表达：

    {(M+m)x¨+mlθ¨cos⁡θ−mlθ˙2sin⁡θ=umlx¨cos⁡θ+43ml2θ¨−mglsin⁡θ=0⇒[M+mmlcos⁡θmlcos⁡θ43ml2][x¨θ¨]=[u+mlθ˙2sin⁡θmglsin⁡θ]

  • 加速度表达式：

    Δ=(M+m)⋅43ml2−(mlcos⁡θ)2
    A−1=1Δ[43ml2−mlcos⁡θ−mlcos⁡θM+m]
    [x¨θ¨]=[43ml2(u+mlθ˙2sin⁡θ)−mlcos⁡θ⋅mglsin⁡θΔ−(mlcos⁡θ)(u+mlθ˙2sin⁡θ)+(M+m)mglsin⁡θΔ]

  • 状态空间形式：

    令 x1=x, x2=x˙, x3=θ, x4=θ˙，则

二. 构建LQR控制器

1. 线性化状态空间表达式

  • 小角度近似：

    sin⁡θ≈θ,cos⁡θ≈1,θ˙2sin⁡θ≈0
    [M+mmlml43ml2][x¨θ¨]=[umglθ]

  • 解为：

    x¨=4u−3mgx34M+m
    θ¨=−3u+3(M+m)gx3l(4M+m)

  • 状态空间表达式：

    x˙=Ax+Bu,x=[xx˙θθ˙]
    A=[010000−3mg4M+m00001003(M+m)gl(4M+m)0],B=[044M+m0−3l(4M+m)]

  • 常用近似（m≪M）：

    A=[010000−mgM0000100(M+m)gMl0],B=[01M0−1Ml]

2. LQR的基本思想

  • 目标：设计反馈控制器，使系统状态 x(t) 尽量接近目标状态（通常为零），同时控制消耗尽量小。

  • 性能指标：

    J=∫0∞[(x−xtarget)TQ(x−xtarget)+uTRu]dt

    • Q：状态加权矩阵（正定或半正定）

    • R：控制加权矩阵（正定）

  • LQR最优控制律：

    u∗=−K(x−xtarget)

    其中 K 由连续代数Riccati方程求解：

    ATP+PA−PBR−1BTP+Q=0
    K=R−1BTP

三. Python实现：倒立摆LQR仿真

1. 项目结构与依赖

  • 倒立摆动力学模型（InvertedPendulum）

  • LQR控制器（LQRController）

  • 仿真器（Simulator）

  • 可视化与动画（matplotlib）

2. 系统建模

  • 状态变量：

    • x1：小车位置

    • x2：小车速度

    • x3：摆杆角度（竖直向上为0）

    • x4：摆杆角速度

  • 动力学方程在InvertedPendulum.dynamics中实现，输入受力u受限于±umax。

  • 数值仿真采用四阶龙格库塔法（RK4）：

    xk+1=xk+dt6(k1+2k2+2k3+k4)

3. LQR控制器设计

  • 对非线性系统在目标点线性化，得到 A,B 矩阵

  • 设定权重 Q,R

  • 求解连续代数Riccati方程，计算反馈增益 K

  • 控制律：u=K(xtarget−x)

4. 仿真与可视化

  • 初始化系统、控制器与仿真器

  • 设定初始状态 x0 与目标状态 xtarget

  • 迭代执行：测量状态（可加噪声）→ 计算控制量 → RK4积分 → 记录数据

  • 使用matplotlib绘制动画与状态曲线，支持导出为GIF
